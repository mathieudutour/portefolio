Optelec, an award-winning project

Realized in JAVA, Optelec is a software to automate the sizing of electrical cables of Bouygues Construction work sites as well as the cost estimate for such an electrical installation.

This project lasted eight months during which I was the only developer. It is now used by all agencies of Bouygues Constuction and received the Engineer 's award of the CNJE which awards the best engineer project realised by a Junior Entreprise in 2013.
